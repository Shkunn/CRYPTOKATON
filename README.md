# CB: CoffeeBean

CoffeeBean or BC for short is a Coffee Themed Crypto Currency.
Money is replace by Coffee to pleasure humans of goof taste.

## Installation

Build CoffeeBean:

```sh
mkdir build
cd build
cmake ..
make
```
