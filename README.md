# Creation of a blockchain without any particular purpose  

This weekend was to create a cryptocurrency. Only the blockchain was created here in C++. The transaction principle is not yet in this github... 

## Installation

Build this Repo:

```sh
rm -rf build
mkdir build
cd build
cmake ..
make
```
